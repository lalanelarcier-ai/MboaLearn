À propos de MboaLearn

MboaLearn est un projet de Mini Learning Management System réalisé dans le cadre du TP de l'UE INF222 - Programmation web. Ce projet démontre les fonctionnalités essentielles d'une plateforme d'apprentissage en ligne en utilisant les technologies HTML, CSS, JavaScript, AJAX, PHP et MySQL.
